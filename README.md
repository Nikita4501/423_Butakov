## Стек технологий
- Язык программирования: C#
- Платформа: WPF (.NET 6.0)
- Библиотека для графиков: System.Windows.Forms.DataVisualization.Charting

## Архитектура приложения
- Главное окно (MainWindow) с навигацией (Frame и три кнопки).
- Три отдельные страницы (Page1, Page2, Page3) для каждого задания.
- Реализованы:
  - проверка заполнения полей и корректности ввода;
  - всплывающие подсказки (ToolTip) для всех элементов;
  - подтверждение выхода из приложения;
  - блокировка полей результата для редактирования;
  - построение графика на третьей странице с использованием WindowsFormsHost.

## Инструкция по запуску
1. Открыть решение в Visual Studio 2022.
2. Восстановить NuGet-пакеты (при необходимости).
3. Запустить проект (F5).
